A B2B-style quick order table for WooCommerce variable products with search and bulk add to cart functionality.

== Description ==

Quick Order Table for Variations transforms your WooCommerce variable product pages into efficient B2B-style order forms. Perfect for wholesale stores, distributors, and any shop that needs to streamline the ordering process for products with multiple variations.

= Key Features =

* **Quick Order Table** - Display all product variations in a clean, easy-to-scan table
* **Bulk Add to Cart** - Add multiple variations to cart with a single click
* **Real-time Search** - Filter variations by SKU, name, or description instantly
* **Flatsome Theme Integration** - Seamless UX Builder element for visual page building
* **Mobile Responsive** - Works perfectly on all devices
* **Vietnamese Language** - Fully translated interface
* **AJAX Powered** - No page reloads, smooth user experience
* **Contact Display** - Show phone number for customer support

= Perfect For =

* Wholesale stores
* B2B e-commerce sites
* Distributors and suppliers
* Electrical/industrial product stores
* Any shop with many product variations

= How It Works =

1. Install and activate the plugin
2. Use the shortcode `[quick_order_table]` on any page
3. Or use the UX Builder element if you have Flatsome theme
4. Customers can search, select quantities, and add multiple variations at once

= Shortcode Usage =

Basic usage (auto-detects product on single product pages):
`[quick_order_table]`

With specific product ID:
`[quick_order_table product_id="123"]`

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to Plugins → Add New
3. Search for "Quick Order Table for Variations"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Go to Plugins → Add New → Upload Plugin
3. Choose the ZIP file and click "Install Now"
4. Activate the plugin

= After Activation =

1. Ensure WooCommerce is installed and active
2. Create or edit a variable product
3. Add the shortcode `[quick_order_table]` to any page
4. Or use the UX Builder element if using Flatsome theme

== Frequently Asked Questions ==

= Does this work with all WordPress themes? =

Yes! The shortcode works with any WordPress theme. The UX Builder integration is a bonus feature specifically for Flatsome theme users.

= Can I use this with simple products? =

No, this plugin is specifically designed for variable products that have variations (like size, color, etc.).

= Does it work on mobile devices? =

Yes! The table is fully responsive and adapts beautifully to mobile screens.

= Can customers add multiple variations at once? =

Yes! That's the main feature. Customers can set quantities for multiple variations and add them all to cart with one click.

= Does it update the mini cart automatically? =

Yes, the plugin uses WooCommerce AJAX to update the mini cart without page reload.

= Can I customize the appearance? =

Yes, you can add custom CSS through your theme's customizer or child theme. All elements use BEM-style class names for easy targeting.

= What about search functionality? =

The plugin includes real-time search that filters products by SKU, name, or description as you type.

= Is it translation ready? =

Yes, the plugin is fully translation ready and comes with Vietnamese translations included.
